XML Documenter

This is a simple tool that allows for the creation of XML based documents from source code. It is used on this site to show the source code of projects. The source code is available on github.

I chose to make this as opposed to using language built-in documentation creation tools because I feel that said documentation tools create a large amount of bloat code, and this way I have more control over what gets outputted, and the formatting of the documentation elements for display.

Roadmap
  1. Implement packages for the Java processor.
  2. Implement imports for the Java processor.
  3. Implement comments for the Java processor.
  4. Implement method reading for the Java processor.
  5. Implement a C# processor with the same functionality as the java processor
  6. Implement a C/C++ processor with the same functionality as the java processor
